# inventory_app

The Inventory App is an application that allows users to create various inventories and manage their items. It is a useful tool for household shoppers who like to keep track of what they have and what needs to be purchased. The application works as an inventory tracker and shopping list, and includes a feature called Receipt Gallery, where users can upload photos of their receipts to keep track of them and organize them in one place.

## Technology Stack

- **Frontend**: HTML, CSS
- **Backend**: Python, Django
- **Database**: SQLite3
- **AI Integration**: OpenAI API
- **Deployment**: Docker, Google Cloud

## Getting Started

1. Clone the repository: `git clone https://github.com/yaseenaskar-git/inventory_app`
2. Install dependencies: `npm install`
3. Set up environment variables: `cp .env.example .env` 
4. Start the app: `npm run dev`
